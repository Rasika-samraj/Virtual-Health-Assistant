# 🩺 Virtual Health Assistant

> **An AI-powered healthcare platform that helps users monitor their health, analyze symptoms, and make informed healthcare decisions through an intuitive and modern web application.**

# 📖 Overview

Virtual Health Assistant is an AI-powered web application developed to simplify access to basic healthcare information. The application enables users to analyze symptoms, receive AI-generated health guidance, and organize health-related information through a clean and user-friendly interface.

The primary goal of this project is to demonstrate how Artificial Intelligence can support users by providing instant health assistance, encouraging preventive healthcare, and improving the overall user experience through modern web technologies.

---

# ✨ Features

* 🤖 AI-powered health assistant
* 🩺 Intelligent symptom analysis
* 💡 Personalized health suggestions
* 📋 Health information assistance
* ⚡ Fast AI responses using the OpenAI API
* 📱 Responsive and user-friendly interface
* 🎨 Modern UI built with Tailwind CSS and shadcn/ui

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Rasika-samraj/Virtual-Health-Assistant.git
```

## Navigate to the Project

```bash
cd Virtual-Health-Assistant
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file in the project root and add:

```env
VITE_OPENAI_API_KEY=your_openai_api_key
```

## Run the Project

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
